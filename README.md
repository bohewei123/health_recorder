# 🏥 慢性健康状况追踪 (Health Recorder)

这是一个基于 Python 和 Streamlit 开发的个人健康追踪应用，旨在帮助用户记录每日身体状况、追踪慢性症状变化趋势，并管理康复训练计划。

## ✨ 主要功能

### 1. 📝 每日健康记录
- **症状追踪**：支持记录多种症状的严重程度（0-10分），包括：
  - 肩颈/背/腰疼痛僵硬
  - 头晕
  - 胃部不适/反流
  - 咽喉不适
  - 干眼症状
  - 疲劳/困倦
- **详细反馈**：可记录具体的感受、可能的诱因以及采取的应对措施。
- **时段记录**：支持按不同时段（早起、上午、中午、下午、晚上）进行记录。

### 2. 🧘 康复训练管理
- **训练反馈**：针对特定的康复动作（如胸锁乳突肌按摩、胸椎伸展等）记录每日练习感受和完成状态。
- **项目管理**：
  - 支持动态添加、删除和重排序训练项目。
  - 初始配置可从 Markdown 模板自动加载。
- **数据导出**：支持将训练反馈导出为格式化的 Markdown 文档，方便归档或分享给理疗师。

### 3. 📈 数据可视化与分析
- **趋势图表**：使用 Plotly 生成交互式图表，直观展示各项症状随时间的变化趋势。
- **历史记录**：查看所有历史记录明细，支持按日期筛选。

### 4. 💾 数据管理
- **本地存储**：所有数据存储在本地 SQLite 数据库 (`health_records.db`) 中，保障隐私安全。
- **数据导出**：
  - 健康记录可导出为 CSV 格式。
  - 训练反馈可导出为 Markdown 格式。

## 🛠️ 技术栈

- **Python 3.8+**
- **Streamlit**: Web 应用框架
- **Pandas**: 数据处理
- **Plotly**: 数据可视化
- **SQLite**: 本地数据库

## 🚀 快速开始

### 1. 环境准备

确保你的电脑上已安装 Python 环境。

### 2. 安装依赖

在项目根目录下运行以下命令安装所需库：

```bash
pip install -r requirements.txt
```

### 3. 启动应用

```bash
streamlit run app.py
```

启动后，浏览器将自动打开应用（通常地址为 `http://localhost:8501`）。

## 📂 项目结构

```
health_recorder/
├── app.py                  # 应用主入口
├── db_manager.py           # 数据库管理模块
├── health_records.db       # SQLite 数据库文件（自动生成）
├── exercise_template.md    # 康复训练项目初始模板
├── requirements.txt        # 项目依赖
└── README.md               # 项目说明文档
```

## 📝 使用指南

1. **初次使用**：
   - 启动应用后，进入"康复训练"页面，系统会自动读取 `exercise_template.md` 初始化训练项目。
2. **日常记录**：
   - 在"每日记录"页面填写当下的身体感受。
   - 在"康复训练"页面打卡当天的训练内容。
3. **查看趋势**：
   - 进入"趋势分析"页面，选择日期范围和关心的症状，查看变化曲线。
